# Abanos

## About

Abanos is a modern multi-vendor e-commerce platform built for the Sudanese market.

The platform connects customers, vendors, couriers, and administrators through a secure, scalable, and data-driven ecosystem.

## Vision

To become the leading digital commerce platform in Sudan.

## Mission

To simplify buying and selling while empowering businesses with technology and data.
